What is HTTP(S)?

Hypertext Transfer Protocol Secure, shortly HTTPS, is an extension of the Hypertext Transfer Protocol. It is used for secure communication over a computer network, and is widely used on the Internet. In HTTPS, the communication protocol is encrypted using Transport Layer Security or, formerly, Secure Sockets Layer. So it greatly increases the security level, protects data transmitted between users from cyber attacks, and ensures the information is not going to be lost in the network.
